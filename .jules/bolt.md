## 2024-05-18 - Mathematical Optimization for SVD of Covariance-Weighted Matrices
**Learning:** In data-aware SVD, computing the square root of the covariance matrix (`Cov_sqrt`) via eigendecomposition to right-multiply it with weights takes `O(n^3)` operations and requires materializing a dense `n x n` matrix. However, `W_weighted = W @ V @ S^{1/2} @ V^T`. By computing `W_tilde = (W @ V) * S^{1/2}` and running SVD directly on it, we get identically correct singular values and left singular vectors. The right singular vectors can then be mapped back via `Vh_orig = Vh_tilde @ S^{-1/2} @ V^T`. This avoids the expensive `n x n` matrix materialization and reduces intermediate matrix multiplications, yielding substantial memory savings and a speedup of >60% for this phase.
**Action:** When working with unitary transformations (like eigenvectors) and eigendecompositions, mathematically trace through the singular value decomposition. Often, the unitary matrix can be commuted or applied after the decomposition to avoid materializing large intermediate matrices.
