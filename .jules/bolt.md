## 2024-05-18 - Avoiding dense covariance square root matrix materialization
**Learning:** When performing data-aware SVD on a matrix $W$ with a covariance matrix $\Sigma_{cov}$, materializing the intermediate dense matrix $\Sigma_{cov}^{1/2}$ and computing $W_{weighted} = W \cdot \Sigma_{cov}^{1/2}$ requires $O(n^3)$ operations and creates a memory bottleneck, which is specifically a problem for scaling to large LLM linear layers. Because eigenvectors are orthogonal/unitary ($U_{cov} \cdot U_{cov}^T = I$), we can bypass materializing $\Sigma_{cov}^{1/2}$.
**Action:** When computing properties that require multiplying by symmetric matrices derived from eigendecomposition (like $\Sigma_{cov}^{1/2}$ or $\Sigma_{cov}^{-1/2}$), project the input matrix into the eigenvector basis first, scale by the eigenvalues, and avoid reconstructing the dense matrix entirely (e.g. $W_{weighted} = (W \cdot U_{cov}) \odot \Sigma_{cov}^{1/2}$).
