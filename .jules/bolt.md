## 2024-04-16 - Data-Aware SVD Dense Intermediate Avoidance
**Learning:** In data-aware SVD algorithms, computing `Cov_sqrt` ($E D^{1/2} E^T$) and multiplying it by the weight matrix ($W$) materializes a large, dense intermediate matrix ($E D^{1/2} E^T$). Because the SVD decomposition $A = U \Sigma V^T$ is invariant to orthogonal transformations on the right side of the input matrix, we can factor out the rightmost $E^T$ from the covariance square root. By using the compact representation $(W E) D^{1/2}$ instead of $W E D^{1/2} E^T$, we can perform SVD and completely avoid materializing both `Cov_sqrt` ($E D^{1/2} E^T$) and `Cov_sqrt_inv` ($E D^{-1/2} E^T$).

**Action:** Whenever working on algorithms that scale features using the square root of a covariance matrix (like data-aware SVD or Whitening), look for opportunities to compute only the necessary weighted projections (e.g. $(WE)D^{1/2}$) and apply the inverse scale ($D^{-1/2} E^T$) directly to the singular vectors instead of using full dense covariance transformation matrices. This yields a mathematically equivalent result with lower memory overhead and fewer matrix multiplications.
