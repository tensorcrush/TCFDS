## 2024-05-19 - O(N^3) Matrix Materialization Avoidance
**Learning:** Dense matrix operations like explicitly forming $N \times N$ covariance square-root matrices (`Cov_sqrt = V @ S^{1/2} @ V^T`) cause severe memory bottlenecks and slow $O(N^3)$ operations for large matrices. When right-multiplying by `Cov_sqrt` prior to SVD, or computing error metrics like Frobenius norm (which is unitarily invariant), we can bypass dense intermediate matrices.
**Action:** When working with covariance decompositions or weighted approximations, always exploit the eigenvalues ($S$) and eigenvectors ($V$) directly. Rely on mathematical properties (unitary invariance, $SVD(A @ V) \implies U, \Sigma, V_A V^T$) to compute the same result without explicitly creating or multiplying $N \times N$ intermediates.
